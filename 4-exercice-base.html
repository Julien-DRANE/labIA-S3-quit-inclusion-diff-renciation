Exercice 1 : créer une base fictive d'élèves

Objectif : partir d'un jeu de données simple et crédible pour construire ensuite des groupes de besoins sans exposer de vraies données d'élèves.

1. Base fictive 2. Créer les groupes 3. Planifier

Jeu de départ

Base fictive prête à l'emploi

La base est déjà structurée pour que les stagiaires puissent passer vite à l'étape de groupement.

Consigne simple

Ce que les stagiaires doivent comprendre

  1. On travaille sur des données fictives, donc sans risque de confidentialité.
  2. La base contient déjà des niveaux de maîtrise Pronote, des réussites, des difficultés et un besoin dominant.
  3. Le but n'est pas de faire de la statistique fine, mais de donner à l'IA une entrée claire pour former des groupes pédagogiquement défendables.

Production stagiaire

Base retenue pour l'exercice

Les stagiaires peuvent garder la base telle quelle ou l'ajuster légèrement avant de passer à l'étape 2.

Astuce : gardez une structure simple, lisible, avec un besoin dominant par élève.
